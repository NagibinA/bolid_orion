# Bolid Orion Protocol для Home Assistant

Интеграция для работы с устройствами Болид по протоколу Orion через MQTT шлюз.

## Возможности

- Автоматическое обнаружение устройств на шине Orion
- Создание сенсоров для каждого найденного прибора
- Отображение типа прибора и версии прошивки

## Установка через HACS

1. Добавьте репозиторий в HACS:
   - Откройте HACS → Интеграции → ⋮ → Пользовательские репозитории
   - URL: `https://github.com/NagibinA/bolid_orion`
   - Категория: Интеграция

2. Нажмите "Установить"

3. Перезапустите Home Assistant

## Настройка

1. Убедитесь, что MQTT шлюз работает и публикует ответы в топик `WemosOrion/answer`

2. Интеграция автоматически подпишется на топик и начнёт создавать сенсоры

## Требования

- Home Assistant 2024.1+
- MQTT брокер
- Шлюз Orion2Mqtt на ESP8266/ESP32

## Поддерживаемые устройства

- Сигнал-20, Сигнал-20П, Сигнал-20М
- С2000-КДЛ, С2000-КДЛ-2И
- С2000-БКИ, С2000-БИ
- С2000-СП1, С2000-СП2, С2000-СП4
- И другие приборы Болид с протоколом Orion
